# 目標管理の使い方

目標管理機能では、財務目標を設定して進捗を追跡できます。

## 目標の種類

- **貯蓄（savings）**: 旅行・家電購入など一般的な貯蓄目標
- **老後資金（retirement）**: 老後のための積立目標（1ユーザー1つまで）
- **緊急資金（emergency）**: 緊急用資金の積立目標（1ユーザー1つまで）
- **カスタム（custom）**: 上記に当てはまらない独自の目標

## 目標の設定方法

1. 「目標管理」ページ（/goals）を開く
2. 「新しい目標」ボタンをクリック
3. タイトル・目標種別・目標金額・目標期日・月間積立額を入力
4. 「作成」ボタンで保存

## 進捗の確認

- 目標ごとに進捗バー（%）で達成状況を確認できる
- 現在の積立額・残り金額・残り日数が表示される
- 「進捗を更新」で現在の積立額を都度入力できる
- アクティブ／非アクティブで目標を有効・無効に切り替えられる

## アクセス方法

ナビゲーションの「目標管理」または /goals

## ポイント

- 目標期日と月間積立額を入力すると達成見込みが自動計算される
- 「推奨事項」機能で目標達成のアドバイスを確認できる
- 「実現可能性分析」で現在の財務状況から達成の難易度を確認できる
