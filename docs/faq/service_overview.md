# FinPlan（財務計画計算機）とは

FinPlanは、日本在住の方向けの個人財務計画ツールです。
将来の資産形成・老後資金・緊急資金の計算、財務目標の管理ができます。

## 主な機能

- **資産推移シミュレーション**: 収入・支出・投資利回りを入力して将来の資産額を予測
- **老後資金計算**: 退職後に必要な資金を年金を考慮して計算
- **緊急資金計算**: 万が一の際に必要な緊急資金の推奨額を計算
- **目標管理**: 貯蓄・老後資金・緊急資金・カスタムの財務目標設定と進捗追跡
- **チャットボット**: 財務に関する質問に回答するAIアシスタント（このBot）
- **PDFレポート生成**: 計算結果を含む財務レポートをPDF形式で出力

## アクセス方法

- トップページから「ゲストとして試す」でログイン不要で全機能を試せる
- アカウント登録するとデータがクラウドに保存され複数端末で同期できる
- 計算機能: /calculations
- 目標管理: /goals
- レポート: /reports
- チャットボット: /bot

## よくある使い方

1. まず「資産推移シミュレーション」で現状の貯蓄ペースを確認する
2. 「老後資金計算」で退職後に必要な資金を把握する
3. 「目標管理」で具体的な財務目標を設定して進捗を追う
4. 「PDFレポート」で家族や専門家と計算結果を共有する
