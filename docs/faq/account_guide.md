# アカウント登録・ログイン・ゲストモードについて

## ゲストモード

- アカウント登録なしでサービスの全機能を試せる
- データはブラウザのローカルストレージに保存（端末をまたいだ共有は不可）
- トップページの「ゲストとして試す」から開始

## アカウント登録

1. トップページの「サインイン / サインアップ」をクリック
2. 登録フォームでメールアドレスとパスワードを入力
3. 登録後、全データがサーバーに保存され複数端末で同期可能になる

## ログイン

- 登録済みのメールアドレスとパスワードでログイン
- パスワードを忘れた場合は「パスワードを忘れた方はこちら」から再設定可能
- 二段階認証（2FA）にも対応

## ゲストモードからアカウントへの移行

- 現状、ゲストデータの自動移行には対応していない
- アカウント登録後は新規データとして入力が必要

## セキュリティ

- パスワードはハッシュ化して保存（平文では保存されない）
- JWT（JSON Web Token）で認証管理
- HTTPS通信で暗号化
