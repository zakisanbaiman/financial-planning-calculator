# 資産推移シミュレーションの使い方

資産推移シミュレーションは、現在の貯蓄ペースで将来どれだけ資産が増えるかを予測する機能です。

## 入力項目

- **現在の資産額**: 現在保有している資産の合計（貯金・投資等）
- **月間積立額**: 毎月積み立てる金額
- **投資利回り**: 年率での期待リターン（例: 5%）
- **インフレ率**: 物価上昇率（例: 2%）
- **シミュレーション期間**: 何年先まで計算するか

## 結果の見方

- グラフで10年後・20年後・30年後の資産推移を確認できる
- インフレ調整後の実質資産額も表示される
- 積立分と運用益の内訳が確認できる

## アクセス方法

ナビゲーションの「計算機能」（/calculations）→「資産推移シミュレーション」をクリック

## ポイント

- 投資利回りは過去の長期平均として株式インデックスは年5〜7%程度が参考値
- インフレ率は日本では近年1〜2%程度が一般的な想定値
- 積立額を変えてシミュレーションすることで「あと月いくら増やせば目標に達するか」がわかる
