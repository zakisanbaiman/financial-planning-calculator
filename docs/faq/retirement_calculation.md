# 老後資金計算の使い方

老後資金計算は、退職後に必要な資金を計算し、現在の貯蓄で達成できるかを確認できる機能です。

## 入力項目

- **現在の年齢**: 計算の基準となる現在の年齢
- **退職予定年齢**: 何歳で退職するか（デフォルト: 65歳）
- **老後の月間生活費**: 退職後に毎月必要な生活費の見込み額
- **老後の生活期間**: 退職後の生活年数（例: 30年）
- **想定年金月額**: 公的年金として受け取れる見込み額

## 結果の見方

- 必要老後資金の総額
- 年金収入を差し引いた自己負担額
- 現在の貯蓄ペースでの達成見込み額
- 充足率（必要額に対して何%達成できるか）

## アクセス方法

ナビゲーションの「計算機能」（/calculations）→「老後資金計算」をクリック

## ポイント

- 年金額の目安は日本年金機構の「ねんきんネット」で確認できる
- 生活費は現役時代の60〜70%が一般的な目安とされる
- iDeCo（個人型確定拠出年金）と組み合わせると節税しながら老後資金を積み立てられる
